KinExercise
=======

An alarm application written in C++ that turn off only when the user completes 3 set of predefined instructions. 

Are you a heavy sleeper and alarm procrastinator?
This is for you!

1.	a. Side bend, left
	b. Side bend, right 
		<br> Repeat 3 times

2.	a. Rise left foot
	b. Rise right foot 
		<br> Repead 3 times

3.	a. Hands up
	b. Hands down knees
	c. Sit up/ Squat and hands up
		Repeat 3 times
